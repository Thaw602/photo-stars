# 双星系开场动画 + 双视角聚焦

## 目标

将当前的"直接展示双星系"改为：
1. 开场两个光球互相绕转
2. 点击光球 → 爆开展开成星系
3. 两个星系可切换焦点（平滑聚焦）

## 架构

```
GalaxyCanvas.tsx               ← Canvas 渲染（纯绘制）
    ↑ props from hook

useGalaxyAnimation.ts          ← 新增：动画阶段机 + 焦点切换
    ↑ reads store

store/index.ts                 ← +3 字段：phase, focusGalaxy, animProgress
```

新增文件 1 个，修改文件 2 个。

## 动画阶段机

| 阶段 | 画面 | 触发 |
|------|------|------|
| `ORBS` | 两光球轨道绕转，暗黑背景 | 页面加载 |
| `EXPAND_GOLD` | 金色光球爆闪→螺旋扩散成星系 | 点金色光球 |
| `EXPAND_BLUE` | 冰蓝光球跟随展开（延迟 0.8s） | GOLD 完成后自动 |
| `DUAL` | 双星系完整，可交互 | 两个都展开后 |

### 展开动画细节

- 爆闪阶段（200ms）：光球瞬间膨胀 3 倍，白光溢出
- 扩散阶段（1.5s）：粒子从中心以 easeOutCubic 飞散到最终旋臂位置
- 光球残影逐渐消散

### 焦点切换（DUAL 内）

- 点击视频星系粒子 → 摄像机 easeInOutCubic 0.8s 滑向视频星系中心
- 点击照片星系粒子 → 滑回来
- 偏移量：`lookAt` 向目标星系偏移，另一个偏到边角

## 视频星系参数调整（姊妹星系）

| 参数 | 旧值 | 新值 | 说明 |
|------|------|------|------|
| `VIDEO_GALAXY_SCALE` | 0.30 | 0.70 | 面积约 50% |
| `VIDEO_GALAXY_Z_OFFSET` | -0.8 | -0.25 | 微深度差 |
| `VIDEO_GALAXY_Y_OFFSET` | +0.25 | -0.45 | 偏下方，对角线构图 |
| `VIDEO_CORE_SCALE` | 0.35 | 0.60 | 配合新比例 |
| `VIDEO_CONNECT_DIST_FACTOR` | 0.55 | 0.70 | 配合新比例 |

## 光球渲染（ORBS 阶段）

- 暗黑背景 + 稀疏深空星点（~200 个）
- 两个光球绕画面中心椭圆轨道运动，周期 6s
- 金色光球：暖色径向渐变，半径 ~8% min(w,h)
- 冰蓝光球：冷色径向渐变，同上
- 轨道半长轴 ~15% min(w,h)，椭圆率 0.3

## 视频粒子填充

视频星系旋臂上额外生成 ~4000 个不可点击装饰粒子：
- 冰蓝色调，0.5-1.5px
- 带微弱正弦闪烁（周期随机 1.5-4s）
- 散布在视频节点周围，填充视觉密度
- 数据存在 ref 中，不进入 PhotoNode 数组

## 渲染流程（DUAL 阶段）

```
1. 清屏
2. 绘制背景星场（共享）
3. 绘制两个星系核心辉光
4. 绘制视频装饰粒子（冰蓝）
5. 绘制主星系旋臂粒子（金色）
6. 绘制视频星系旋臂粒子（冰蓝）
7. 视频星座连线（仅视频星系内）
8. 深度排序 + 绘制所有节点（照片 + 视频菱形）
9. 行星（如有）
10. 焦点偏移过渡
```

## 焦点切换过渡

- 维护当前 `focusTarget: { x, y, z }` 和 `currentFocus: { x, y, z }`
- 每帧 `currentFocus` 向 `focusTarget` lerp（lerpFactor 0.08）
- 焦点切换时触发 easeInOutCubic 计时器 0.8s
- 相机 project3D 时加入 currentFocus 偏移
- 非聚焦星系降低透明度 20% 强化层次
